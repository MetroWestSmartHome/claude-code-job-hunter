## Tips for Tracking

**Status Meanings:**
- **Discovery Pipeline:** Job scored/found, needs review before applying
- **In Progress:** Creating/customizing resume and cover letter
- **Submitted:** Application sent, waiting for response
- **Interviews Scheduled:** Active interview process
- **Closed:** Final outcome reached (rejected, offer accepted, or withdrawn)

**What to Track:**
- All applications (even if rejected quickly - helps identify patterns)
- Source (job-hunter score helps validate scoring accuracy)
- Salary range (ensures you're targeting right compensation level)
- Gaps and how addressed (improves future applications)
- Timeline (how long from application to response)

**Review Regularly:**
- Weekly: Check active applications, follow up if needed
- Monthly: Analyze closed applications for patterns
- Quarterly: Update candidate profile based on feedback
