## Guidelines for Writing Bullets

**Action Verbs:** Led, Architected, Implemented, Managed, Drove, Designed, Developed, Established, Spearheaded, Collaborated, Built, Optimized, Automated, Migrated, Deployed

**Structure:**
- Start with action verb
- Include specific technologies/tools used
- Quantify impact when possible (%, $, time saved, users affected)
- Focus on outcomes, not just tasks

**Good Examples:**
- ✅ "Architected event-driven automation platform using Workflow Automation Platform reducing manual onboarding time by 75%"
- ✅ "Led enterprise-wide EDR Solution deployment across 500+ Windows, Mac, and iOS devices"
- ✅ "Managed IT budget of $2M with vendor negotiations achieving 15% cost reduction"

**Bad Examples:**
- ❌ "Responsible for EDR Solution" (not specific enough)
- ❌ "Did some Cloud Platform work" (vague, no action verb, no outcome)
- ❌ "Helped with security projects" (weak verb, no specifics)

---

**Tips:**
1. Add bullets as you accomplish new projects
2. Keep comprehensive - don't remove bullets, just add
3. Organize by category for easy selection
4. Include technologies even if not used recently (shows breadth)
5. Quantify when possible, but qualitative impact is fine too
